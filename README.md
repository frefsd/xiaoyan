# 🌌 斗破苍穹 · 主题网页

> “三十年河东，三十年河西，莫欺少年穷！”  
> —— 一部致敬经典玄幻小说《斗破苍穹》的静态网页

这是一个使用 **HTML + CSS** 构建的静态网页，灵感来源于天蚕土豆的玄幻小说《斗破苍穹》。通过精美的排版与设计，展现了小说中的世界观、主要角色与经典语录，向这部陪伴无数人成长的经典作品致敬。

---

## 📖 项目简介

本项目是一个多页面的静态网页，展示了：

- 斗破苍穹的世界观设定（斗气大陆、等级体系）
- 主要角色介绍
- 经典语录摘录
- 精美背景与动态效果

> ✅ 纯前端实现，无后端依赖，开箱即用！
